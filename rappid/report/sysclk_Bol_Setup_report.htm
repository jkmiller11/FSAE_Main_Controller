The value for system Clock Frequency is,
System Clock Frequency = 16MHz IRC

System Clock Frequency = 16.000 MHz



Progressive Clock Switching Disabled and Input Division Factor(IDF) set to
2 ,Output Division Factor(ODF) Set to 4 and Loop Division Factor (LDF) set to 64

Final Reference Frequency Multiplier = LDF / (IDF * ODF)
= 64/(2*4)
= 8.000

PLL Output Frequency = XOSC Frequency * Final Reference Frequency Multiplier
= 16.000 * 8.000
= 64.000 MHz

VCO Frequency = XOSC Frequency * (LDF / IDF)
= 16.000 * ( 64 / 2 )
= 256.000 MHz

Frequency Modulation is Disabled with Spread Type set to Center Spread and the Modulation Period and Increment Step are set to 0 and 0 respectively.

In Clock Output Setup, Clock Output is Disabled and Clock Output Source is set to Crystal Oscillator Clock Divider is set to 1.

Clock Output frequency = Clock Output Source /Clock Divider
= 8.000 MHz

Output Clock Pin : = 16 (PA0) - Not Assigned

For Peripheral Clock Setup,

Divider for Set 1(LINFlex & I2C) is Enabled with Divider as 1

Divider for Set 2(FlexCAN & DSPI) is Enabled with Divider as 1

Divider for Set 3(eMIOS,ADC & CTU) is Enabled with Divider as 1

.Final Clock for all the three peripheral sets is as follows,

Set 1 (LINFlex & I2C)
Final Clock = System Clock Frequency / Divider
= 16.000/1
= 16.000 MHz

Set 2 (FlexCAN & DSPI)
Final Clock = System Clock Frequency / Divider
= 16.000/1
= 16.000 MHz

Set 3 (eMIOS,ADC & CTU)
Final Clock = System Clock Frequency / Divider
= 16.000/1
= 16.000 MHz
