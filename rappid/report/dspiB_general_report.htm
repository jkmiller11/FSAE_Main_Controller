Device Settings for: DSPI_1

General Configuration

DSPI module is Enabled by running it's clock. It is in Master mode and it is configured as SPI (Serial Peripheral Interface.) In debug mode it does not halt serial transfer. Modified SPI transfer format is disabled. Transmit FIFO is enabled and receive FIFO is enabled. Incoming data is ignored once receive FIFO is full. Sample point is 0. Continuous serial communication clock is disabled and DSPI transfer is started.The Transfer count at reset is set to 0

Peripheral chip select strobe is disabled and chip selection details are as below:
