Device Settings for: General Configuration



General Configuration

There are total eleven modes and different configuration for each mode is as explained below:

1) RESET Mode :–

a. RESET Mode is Enabled.
b. I/O output power down control is disabled means pads power sequence driver is enabled.
c. Main voltage regulator is switched ON
d. Data Flash is in Normal mode.
e. Code Flash is in Normal mode.
f. System PLL is switched OFF
g. Crystal Oscillator is switched OFF
h. 16 MHz Internal RC oscillator is switched ON
i. System clock is selected as 16MHz internal RC oscillator


2) TEST Mode :–

a. Test Mode is Disable.
b. I/O output power down control is disabled means pads power sequence driver is enabled.
c. Main voltage regulator is switched ON
d. Data Flash is in Normal mode.
