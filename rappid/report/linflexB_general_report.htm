Device Settings for: LINFlex_1

General Configuration

LINFlex module configured in Slave Mode
Loopback Mode is disabled, Self Test Mode is disabled
Receive Buffer not locked on overrun
Break length in Master mode is 10 bits
Slave Mode Break Detection Threshold is 11-bit break
Rx interrupt generated if ID does not match any filter.
The Sleep mode is exited on software request
Checksum field is sent after the required number of data bytes is sent.
Checksum calculation is Disabled
LIN Slave Automatic Resynchronization is disabled
Idle on Bit Error : Bit error reset LIN state machine

Idle on Identifier Parity Error : Identifier Parity error reset LIN state machine.
Program checksum is 0b0

Integer Baud Rate Factor = 0
Fractional Baud Rate Factor = 0

Baud Rate (Symbols/sec) = Fcpu/(16*LFDIV)
Where , LFDIV = (Mantissa + Fraction) = 0.0+0.0/16=0.0
Baud Rate (Symbols/sec) = 1.6E7 /(16*0.0) = Infinity
