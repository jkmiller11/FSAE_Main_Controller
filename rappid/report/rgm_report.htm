Device Settings for: RGM

Overview

RGM module provides the register interface to monitor and control the device reset sequence. Module centralizes the different reset sources and configures them to Enable or Disable the system reset. The reset sequencer is a state machine having different phases (PHASE0, PHASE1, PHASE2, PHASE3, and IDLE). RGM also enables to have complete or short reset sequence for individual reset sources.

Destructive Event
1. 2.7V low-voltage detected

Reset sequence is set to trigger a reset sequence
Alternate Request type set to generate a SAFE mode request event if the reset is disabled

2. Software watchdog timer

Reset sequence is set to trigger a reset sequence
Alternate Request type set to generate a SAFE mode request event if the reset is disabled

3. 1.2V low-voltage detected (power domain #1)
Reset sequence is set to trigger a reset sequnce
Alternate Request type set to to generate a SAFE mode request event if the reset is disabled

4. 1.2V low-voltage detected (power domain #0)
Reset sequence is set to trigger a reset sequence
Alternate Request type set to generate a SAFE mode request event if the reset is disabled

Functional Event
1. Code or data flash fatal error

Reset sequence is set to trigger a reset sequence
Alternate Request type set to generate a SAFE mode request event if the reset is disabled
ShortSequence reset is set to Start From PHASE1
Bidirectional Reset is set to External Reset asserted

2. 4.5V low-voltage detected

Reset sequence is set to trigger a reset sequence
Alternate Request type set to generate a SAFE mode request event if the reset is disabled
ShortSequence reset is set to Start From PHASE1
Bidirectional Reset is set to External Reset asserted

3. CMU0 clock frequency higher/lower than reference

Reset sequence is set to trigger a reset sequence
Alternate Request type set to generate a SAFE mode request event if the reset is disabled
ShortSequence reset is set to Start From PHASE1
Bidirectional Reset is set to External Reset asserted

4. Oscillator frequency lower than reference

Reset sequence is set to trigger a reset sequence
Alternate Request type set to generate a SAFE mode request event if the reset is disabled
ShortSequence reset is set to Start From PHASE1
Bidirectional Reset is set to External Reset asserted

5. PLL0 fail

Reset sequence is set to trigger a reset sequence
Alternate Request type set to generate a SAFE mode request event if the reset is disabled
ShortSequence reset is set to Start From PHASE1
Bidirectional Reset is set to External Reset asserted

6. Core reset

Reset sequence is set to trigger a reset sequence
Alternate Request type set to generate a SAFE mode request event if the reset is disabled
ShortSequence reset is set to Start From PHASE1
Bidirectional Reset is set to External Reset asserted

7.JTAG initiated reset

Reset sequence is set to trigger a reset sequence
Alternate Request type set to generate a SAFE mode request event if the reset is disabled
ShortSequence reset is set to Start From PHASE1
Bidirectional Reset is set to External Reset asserted
