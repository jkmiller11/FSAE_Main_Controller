Device Settings for: I2C_0 Config
I2C System Clock frequency is 16 MHz. I2C Module is Disabled
The address of the Slave that wishes to communicate with controller using I2C is set to 0

I2C Baud rate, SDA Hold Value, SCL Start Hold Time and SCL Stop Hold time are
calculated using I2C Clock Rate which is set to 16.0 MHz

SCL Divider, SDA Hold Value, SCL Start Hold Value and SCL Stop Hold value are
calculated using I2C Clock Rate. There values are shown in table shown below.

Baud rate = Bus Clock Speed / (SCL Divider)
Baud rate = 16.0* 10 ^-6 /(20) = 0.800000 MHz

SDA hold Time = Bus Clock Period * SDA Hold Value
SDA hold Time = 1/16.0* 10 ^-6 *(7) = 0.437500 *10 ^-6 Seconds

SCL Start Hold Time = Bus Clock Period * SCL Start hold value
SCL Start Hold Time = 1/16.0* 10 ^-6 *(6) = 0.375000 *10 ^-6 Seconds

SCL Stop Hold time = Bus Clock Period * SCL Stop hold value
SCL Stop Hold time = 1/16.0* 10 ^-6 *(11) = 0.687500 *10 ^-6 Seconds

Bus clock Multiplier 1.I2C is in Slave and Receive mode. I-Bus Interrupt is Disabled.
and Bus Idle Interrupt is Disabled. Data Acknowledge disable bit is set to Acknowledge.
I2C Bus Master will Not Generate repeat start cycle.
