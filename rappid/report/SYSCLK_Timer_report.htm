Device Settings for: SWT

SWT Configuration


SWT is enabled. with SWT Time-out register, SWT_TO is running on Oscillator Clock IRC 128 KHz. SWT is configured such that counter is stopped in debug mode and continues to run in STOP mode. SWT action on timeout is reset on a time-out and operating in In Regular mode (non window mode) the servicing sequence should be executed any time before the Regular Mode (not window mode). 1280 clock cycles (10 ms) is completed.

Reset on Invalid Access is Enabled.
Soft Lock is Disabled and Hard Lock is Disabled.
Watchdog Timeout Range: 10 ms to 33554.43 sec
Watchdog Resolution: 0.008 ms

Watchdog Timeout = (Watchdog Timeout Value / 128) ms
Watchdog Window Time = (Window Value / 128) ms
Watchdog Resolution = (1/128) ms
Window Time % = 100 * (Window Value / Watchdog Timeout Value)
