The details for CMU0 parameters are as follows,

CMU0

With reference to parameter table above, the High & Low Frequency Reference Value in frequency units is given by following formulas

High Frequency Reference = (HFREF_A[11:0]/16) * (FRC/4)
=(4095/16)*(16/4)
High Frequency Reference = 1023.75 MHz

Low Frequency Reference = (LFREF_A[11:0]/16) * (FRC/4)
=(0/16)* (16/4)
Low Frequency Reference = 0.0 MHz

The IRC Clock Divided Frequency is given by following formula,
IRC Clock Divided Frequency = CK_IRC/2^RCDIV
= 16/2^3
IRC Clock Divided Frequency = 2.0 MHz
