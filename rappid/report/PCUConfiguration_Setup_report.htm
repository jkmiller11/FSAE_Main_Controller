Device Settings for: PCU

PCU Configuration

The power control unit is used to reduce the overall device power consumption. Power can be saved by disconnecting parts of the device from the power supply via a power switching

Only Power Domain # 2 is user configurable. By setting corresponding bits in PCONF2 register it is possible to switch on all or only the first 8KB SRAM in respective modes.

Currently 8KB SRAM is available in STBY0 .
