SPI Commands
SPI Command has following parameters:
CONT: Continuous Peripheral Chip Select.
CTAS: Clock and Transfer Attributes Select.
EOQ: End of Queue.
CTCNT: Clock Phase and Data Capture
TxDATA: Data Transfer Order
