Device Settings for: System Setup

FlashBIU


Bank Configuration

Flash Bank 0: Code is configured with 3 cycle Read wait state and 3 cycle Write wait state. 3 hold Cycles are required between flash array access requests. Buffer is Enabled. Data Prefetch is Disable. Instruction Prefetch is Enable.
The Prefetch limit is set to Prefetch on miss or hit. The Page Buffer Configuration is set to 3 instruction + 1 data.

Flash Bank 1: Data is configured with 3 cycle Read wait state and 3 cycle Write wait state. 3 hold Cycles are required between flash array access requests. Buffer is Enabled.
