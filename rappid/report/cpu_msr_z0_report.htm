Device Settings for: System Setup

SPR




Wait State is disabled. Critical Interrupt is disabled. External Interrupt Enable is disabled. Problem State is disabled. Machine check is disabled. Debug Interrupt Enable disabled. Instruction address space is disabled. Data address space is disabled
Recoverable Interrupt is disabled

External input interrupt will remain unaffected by the debug interrupt. Critical interrupt will remain unaffected by the debug interrupt. The debug interrupt will remain unaffected by the critical interrupt but it will remain unaffected by the machine check interrupt. Debug interrupt use the critical interrupt resources CSRR0/CSRR1 for saving state.Interrupt Inputs Clear disabled. Machine Check Exception is disabled.
